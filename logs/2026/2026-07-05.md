### Memory Bridge v1.2.0
Unified MCP server bridging three memory dimensions:
- **Concept Graph** (Mind) — spreading-activation recall with Ebbinghaus forgetting
- **Semantic Vector** (Honcho) — pgvector HNSW search with local nomic-embed-text-v1.5
- **Episodic Memory** — JSONL session timeline with auto cross-system storage

**Cortex Accelerator:** LLM-accelerated knowledge synthesis across all memory systems, clustering related memories and generating consolidated summaries. Uses local Ollama for synthesis; falls back to extractive key-sentence selection when unavailable. Includes contradiction detection (negation heuristic + LLM validation).

8 MCP tools. 51/51 tests green. Zero dependencies beyond Python stdlib. Fully offline.

### MCP Tray Manager
Windows system tray app showing real-time health of all MCP services. Python daemon probes 7 services every 30 seconds (TCP, HTTP, and file-based checks). AHK v2 tray menu shows [UP]/[DOWN] status with detail strings. One-click daemon launcher from tray. Installed to Windows startup. 7/7 tests green.

### Honcho Docker Stack Recovery
Recovered the local Honcho semantic memory stack. Full volume wipe and fresh provisioning. Fixed pgvector dimension mismatch (1536 → 768 for nomic-embed-text-v1.5). API healthy, all containers running with restart policy.

### Speak-to-Type
Offline speech-to-text dictation — the inverse of the previously open-sourced Read Aloud TTS. Uses faster-whisper (tiny.en model, ~75MB) for local transcription. Same architecture as Read Aloud: AHK v2 tray app + Python daemon with file-based JSON protocol. Ctrl+Shift+V toggles recording, silence auto-stops after 2 seconds, text types into active window. Clipboard-safe. Installed to Windows startup. 7/7 tests green. Zero cloud.

### Screen Context Provider
MCP server giving AI agents eyes on the user's Windows workspace. Uses Win32 API via ctypes (zero external Python dependencies) to capture active window title, process, class, and position. Enumerates all visible top-level windows with optional title filter. Agents can answer "what are you looking at?" without user description. 2 MCP tools. 5/5 tests green.
